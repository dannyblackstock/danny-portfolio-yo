City Impact

mobile web + icon design

Client
Sense Tecnic Systems
Year
2013
Role
Redesign icons and interface elements for a mobile web application.
Tools
Illustrator, Photoshop, Kurugo, CSS

I volunteered to create the graphics for a concept mobile application, called City Impact, being developed by Sense Tecnic, a small Vancouver software company.

My tasks for this project included creating improved icons, as well as interface elements like menus and buttons. The icons and elements I created were used in demo applications for the cities of Richmond and Surrey, British Columbia.

Icon design concepts.

My designs were based on icons that had previously been used, but I also aimed to keep consistent with the existing Richmond and Surrey branding and website design. I then implemented my designs into the web application, by creating a theme for the Kurugo mobile framework, which the application is built on.

The original application design, left, and my mockups for Richmond and Surrey.
The original user interface, left, compared to my mockups for Richmond.

Various demos of my designs in action are now viewable on the City Impact Beta website, which is now a part of the Urban Opus project. I also took the photo used in the City Impact banner.

Some of my icons were also implemented into Android and iOS applications.
